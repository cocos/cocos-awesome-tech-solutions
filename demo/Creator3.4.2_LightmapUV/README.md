
# 简介
基于 CocosCreator 3.4.2 版本创建的**检测模型是否有2uv**扩展。

# 背景
生成光照贴图，模型除了本身的 UV，还需要另外包含一套 UV，详情参考：https://docs.cocos.com/creator/manual/zh/concepts/scene/light/lightmap.html，可以通过该拓展检测缺少2uv的模型
  

# 使用说明

  > 在拓展中添加check2uv

  >
